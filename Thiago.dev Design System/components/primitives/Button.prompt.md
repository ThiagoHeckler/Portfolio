Primary/secondary/ghost action button — amber fill for the main CTA, outline for the alternate, on the dark theme.

```jsx
<Button href="#projetos">Ver projetos</Button>
<Button href="https://github.com/ThiagoHeckler" variant="secondary">GitHub</Button>
```

Variants: `primary` (amber), `secondary` (outline), `ghost` (text). Sizes: `sm` / `md` / `lg`. Renders `<a>` when `href` is set, else `<button>`.
